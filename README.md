# 國立臺北教育大學｜社會領域 AI 教學 APP 入口網

整合 21 款 AI 教學應用、QR Code、主題分類與教學圖像，協助教師快速找到合適的數位教學工具。

## 公開網站

**[立即開啟 AI 教學 APP 入口網](https://ntue-social-ai-apps.janeykases.chatgpt.site)**

網站已設定為公開，任何人持網址皆可瀏覽，無須登入。

## 主要特色

- 21 款教學 APP 集中呈現
- 每款 APP 提供可掃描 QR Code 與可愛小圖示
- 科技感視覺背景與 Q 版教學圖像
- 支援手機、平板及桌面瀏覽器
- 可依教學主題快速瀏覽工具

## 專案技術

本專案使用 Vinext 與 React 建置，並透過 OpenAI Sites 發布完整動態網站。
